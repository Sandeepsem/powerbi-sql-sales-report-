## 🎯 Objective

The objective of this project is to design and implement a **dynamic and interactive Sales Performance Dashboard** that provides actionable insights into:

* Revenue Growth 📈
* Profitability 💰
* Sales Patterns 📊
* Regional & Product Performance 🌍

---

## ⚙️ Project Workflow

### 1️⃣ Data Import

* Imported data from Excel files
* Ensured correct formats and data types
* Connected dynamic data source for refresh

---

### 2️⃣ Data Cleaning & Transformation

* Extracted Year, Month, Day, Quarter
* Merged input & master data
* Created relationships with Date table
* Removed duplicates & handled null values
* Validated data accuracy

---

### 3️⃣ Data Modelling (DAX)

#### 📌 Calculated Columns

* Total Cost = Quantity × Standard Cost
* Total Selling = Quantity × Price × (1 – Discount %)
* Discount Amount = Price × Discount %

#### 📊 Measures

* Profit = Selling – Cost
* Margin % = Profit / Selling
